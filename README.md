# DecodeLabs Frontend Project 2

This project is a responsive web layout built for DecodeLabs Frontend Development Project 2. It uses a mobile-first approach and demonstrates the core skills needed for responsive design:

- CSS media queries
- responsive navigation
- fluid spacing and alignment
- CSS Grid and Flexbox layout
- accessible, touch-friendly controls

## Features

- Mobile-first responsive design
- Hamburger-style navigation for small screens
- Fluid typography and spacing using flexible units
- Clean sections for About, Skills, Resources, and Contact
- Easy preview with a local static server

## How to run

1. Open the project folder in your terminal.
2. Start a local server:

   python3 -m http.server 3000

3. Open your browser and visit:

   http://127.0.0.1:3000/

## Project files

- index.html — main page structure
- styles.css — responsive styling and layout
- script.js — mobile navigation interaction

## Goal

The goal of this project is to create a simple, professional responsive webpage that works well across different screen sizes, from mobile phones to desktops.
